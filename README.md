## 📌 Project Overview

The **Ola Ride Analytics Dashboard** is an interactive Power BI project designed to analyze ride booking data and provide actionable business insights. The dashboard enables users to monitor booking performance, revenue trends, vehicle utilization, cancellation analysis, and customer ratings through dynamic visualizations.

This project demonstrates end-to-end business intelligence skills including data cleaning, data modeling, DAX calculations, KPI creation, and interactive dashboard design.

---

## 🎯 Project Objectives

- Analyze overall ride booking performance.
- Monitor revenue generated from different ride categories.
- Evaluate vehicle type utilization.
- Identify cancellation trends and their impact.
- Analyze customer and driver ratings.
- Build an interactive dashboard for business decision-making.
